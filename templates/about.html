About This Site

Hi! My name is Irene, this is my first Django practice project for a book e-commerce store. I use Beautiful soup to scrape the data from toscrape.com.

Backend and frontend are Django python

Last update: 7 June 2021

Some current features:

  • Django Book Object
  • Create a book with form, send the data using API to the backend
  • Upload an excel to bulk update books
  • Add book to basket using cookies

Upcoming features:

  • Recommender system (content based)
  • User can create account
  • Web analystics implementation (Traffic origin, CTR, web behaviour) without using GA
  • User generated content (reviews)
